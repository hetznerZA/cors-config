# Cors Config

Middleware that allows you to configure CORS via a YAML file.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'cors-config'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install cors-config

## Usage

In config.ru

```
use Cors::Config
```
